# Прогноз концентрации продукта печи

## Описание задачи
Необходимо предсказать концентрацию выходного продукта на основе телеметрии печи.

Лабораторные измерения приходят с задержкой 10–15 минут и нерегулярно, поэтому модель используется как soft sensor для оперативной оценки качества.

---

## Данные
- data_train.csv — телеметрия (минутная)
- target_train.csv — лабораторные замеры
- data_test_small.csv — тестовая телеметрия
- target_test_small.csv — тестовые замеры

---

## Решение

### Основные шаги:
1. Анализ данных (EDA)
2. Синхронизация телеметрии и target с учетом задержки
3. Feature engineering:
   - лаги
   - rolling-статистики
   - производные
4. Обучение моделей:
   - Ridge Regression
   - Random Forest
   - Gradient Boosting
5. Оценка качества:
   - MAE, RMSE, MAPE, WAPE
   - Direction Accuracy
6. Анализ остатков
7. Интерпретация признаков
