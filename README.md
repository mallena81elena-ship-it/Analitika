# Прогноз концентрации продукта печи

## Лабораторная работа №2
----
### «Виртуальный датчик для контроля процесса обжига в печи»
Клиент - крупное промышленное предприятие, использующее на одном из этапов производства печь для обжига сырья. Качество выходного продукта этого этапа зависит от ряда параметров работы печи, которые подстраиваются оператором в зависимости от показаний установленных в печи датчиков, а также результатов лабораторных замеров.
Один из ключевых показателей процесса обжига - концентрация выходного продукта - измеряется в лаборатории по следующему процессу:
Оператор по расписанию производит изъятие проб из печи,
Передает пробы на анализ в лабораторию,
Лаборатория поставляет результаты анализов примерно спустя 10-15 минут после изъятия проб.
Лаборатория производит определение целевого показателя качественно, но с учетом задержки в получении результата анализа и человеческого фактора (риск задержки или пропуска взятия пробы оператором) оператор узнает значение показателя с большой задержкой и нерегулярно.


В случае возникновения в печи нештатной ситуации, требующей корректировки параметров обжига, оператор рискует внести необходимые корректировки в параметры процесса слишком поздно, что приведет к выпуску некачественной продукции.

## Решение

### Основные шаги:
1. Анализ данных (EDA)
2. Синхронизация телеметрии и target с учетом задержки
3. Feature engineering:
   - лаги
   - rolling-статистики
   - производные
4. Обучение моделей:
   - Ridge Regression
   - Random Forest
   - Gradient Boosting
5. Оценка качества:
   - MAE, RMSE, MAPE, WAPE
   - Direction Accuracy
6. Анализ остатков
7. Интерпретация признаков
